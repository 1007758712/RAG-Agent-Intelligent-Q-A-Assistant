# RAG + Agent 智能问答助手
## 项目介绍
本项目是一个基于 **检索增强生成（RAG）+ 智能决策体（Agent）** 的智能问答系统，支持文档上传、知识库检索、多轮对话、上下文理解、自动工具调用等能力。  
系统能够**自主判断**用户问题是否需要检索知识库，并自动调用 RAG 工具获取精准答案，有效避免大模型幻觉，提升回答可靠性与专业性。

适用于企业内部知识库、文档助手、智能客服、学习助手等场景。

## 核心功能
- 支持 PDF / TXT 文档上传与解析
- 文档自动切分、向量化、向量库存储
- 基于 RAG 的精准检索问答
- 集成 Agent 智能决策，自动判断是否调用工具
- 多轮对话 + 历史记录管理
- 上下文理解 + 模糊问题自动补全
- 支持本地模型推理，降低 API 成本
- 简洁友好的交互界面

## 技术栈
- 语言：Python  
- 框架：LangChain、LangGraph  
- 大模型：通义千问 / 本地模型（Trae）  
- 向量库：Chroma   
- 界面：Streamlit  
- 部署：Docker、FastAPI  
- 其他：Embedding、Prompt 工程、会话管理

## 项目架构
1. **文档处理层**：文档加载 → 文本切分 → 向量化 → 向量库存储  
2. **检索层（RAG）**：用户问题 → 向量检索 → 获取相关上下文  
3. **决策层（Agent）**：判断问题类型 → 选择是否调用 RAG  
4. **对话层**：多轮历史管理 → 问题补全 → 生成最终回答  
5. **接口层**：提供 Web 界面与 API 服务

## 快速开始
### 1. 克隆项目
```
git clone https://github.com/1007758712/RAG-Agent-Intelligent-Q-A-Assistant.git
cd rag-agent-assistant
```

### 2. 安装依赖
```
pip install -r requirements.txt
```

### 3. 配置环境变量
创建 .env 文件，配置：
```
API_KEY = "your-api-key"
BASE_URL = "your-api-url"
```

### 4. 启动项目
```
streamlit run app.py
```
